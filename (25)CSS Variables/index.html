CSS variables, also known as custom properties, allow you to define reusable values in your stylesheet. They make your code cleaner, easier to maintain, and allow for dynamic styling changes.
  • Use the -- prefix to define a variable.
  • Typically defined in the :root selector to make them available globally.
  • Use the var() function to access the variable's value.
